# Projeto SQL - Restaurante

Simulação de um banco de dados para um restaurante fictício, desenvolvida como parte dos meus estudos em análise de dados.

## Estrutura do projeto:

- create_tables.sql → criação das tabelas
- inserts.sql → inserção de dados
- consultas.sql → consultas SQL (SELECT)
- updates.sql → atualizações e inserção de dados

## Objetivo:

Praticar conceitos fundamentais de SQL aplicados a um cenário realista de negócio.
